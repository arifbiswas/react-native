Welcome to React Native Library

This repository contains useful components and code snippets for React Native development.

Project Structure

  • NewProjectSetup: Initial setup files for starting new projects.
  • set_deeplinking: Implementation of deep linking functionality.
  • vision pay implementation: Payment integration for vision systems.
  • AddGiftChart.jsx: Component for managing gift charts.
  • DataLoadMoreHelper.jsx: Helper utility for data pagination.
  • NativeFirebase.jsx: Firebase integration with native modules.
  • PdfGenerator.jsx: Component for generating PDFs.
  • Voice_To_Text.jsx: Component for voice-to-text functionality.
